This project features a sleek Hidden Search Widget that expands from a simple icon into a full input field upon clicking, optimizing space and aesthetics.

### Key Learnings:
* **Smooth Transitions:** Mastering CSS properties to create fluid expansion animations.
* **Class Manipulation:** Using JavaScript `classList.toggle()` to trigger UI states.
* **Auto-Focus Logic:** Ensuring the input field is focused automatically when expanded for a better user experience.
* **Interactive UI:** Designing modern components that respond intuitively to user actions.
